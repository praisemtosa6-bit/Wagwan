WAGWAN – Full Platform Documentation
A Malawi-first streaming platform built to allow viewers to subscribe and gift their favorite creators using local payments.

1. Overview
Wagwan is a live-streaming and creator monetization platform built for the Malawian and African market. The platform solves the biggest problem African streamers face: they cannot monetize effectively on global platforms because local currencies and payment systems are not supported.
Wagwan provides:
	•	Local subscriptions using PayChangu, Airtel Money, TNM Mpamba
	•	Virtual gifting (coins → gifts)
	•	Creator payouts in MWK
	•	Affordable viewer-friendly pricing
	•	Streamer-first revenue split
Wagwan is NOT built to compete with Twitch on advanced features. The mission is simple, local, and profitable for creators.

4. Monetization System
Wagwan prioritizes simple and predictable revenue models.
4.1 Subscriptions
Three tiers:
	•	Basic – MK 1,500
	•	Super Fan – MK 3,000
	•	VIP – MK 7,000
Creator Earns: 75% Platform Earns: 25%
4.2 Coins (Virtual Currency)
Buy coin packs using PayChangu:
	•	MK 300 → 30 coins
	•	MK 1,000 → 120 coins
	•	MK 2,500 → 350 coins
	•	MK 5,000 → 800 coins
	•	MK 10,000 → 1,800 coins
Gifts → convert coins to MK for creators
Creator Earns: 65% of gift value Platform Earns: 35%
4.3 PPV (Pay-Per-View)
Creators can host special paid shows:
	•	MK 500
	•	MK 1,000
	•	MK 2,000
Creator Earns: 70% Platform Earns: 30%
4.4 Ads (Optional)
	•	Banner ads
	•	Pre-roll
Revenue split: 50/50

6. Architecture Overview
6.1 Tech Stack
	•	Frontend: React Native (mobile), React (web)
	•	Backend: Node.js (NestJS or Express)
	•	Database: PostgreSQL (via Supabase or RDS)
	•	Streaming: Livepeer (cheap, scalable) + Cloudflare CDN
	•	Auth: Supabase Auth
	•	Storage: S3-compatible
6.2 System Diagram
	1	User interacts with app → backend API
	2	Backend interacts with DB and streaming provider
	3	Payments handled by PayChangu
	4	Webhooks update wallet balances and transactions
